# Dify 后端 DSL 导入校验链分析

> 基于 dify-test 源码 `api/services/app_dsl_service.py` 及各 ConfigManager  
> 分析日期: 2026-04-08

## 一、导入总流程

```
import_app()
  ├─ 解析 YAML（语法检查）
  ├─ 检查 DSL 版本兼容性（CURRENT_DSL_VERSION = "0.6.0"）
  ├─ 检查 app 段存在性
  ├─ 检查 app_id（如提供）
  ├─ 版本冲突 → PENDING 状态，存 Redis，等用户确认
  ├─ 提取依赖列表（dependencies）
  └─ _create_or_update_app()
      ├─ 解析 app_mode
      ├─ 创建/更新 App 记录
      ├─ 如果 WORKFLOW / ADVANCED_CHAT:
      │   └─ WorkflowService.sync_draft_workflow()
      │       → 验证 graph 结构、节点数据、边连接、变量引用
      └─ 如果 CHAT / AGENT_CHAT / COMPLETION:
          └─ XxxAppConfigManager.config_validate()
              → 按链式调用各 Manager 校验+设默认值
```

**关键常量:**
- `DSL_MAX_SIZE = 10 * 1024 * 1024`（10MB）
- `IMPORT_INFO_REDIS_EXPIRY = 10 * 60`（10 分钟）
- 导入状态枚举: `COMPLETED` / `COMPLETED_WITH_WARNINGS` / `PENDING` / `FAILED`

## 二、Workflow & Chatflow（Graph-Based）校验

调用 `WorkflowService.sync_draft_workflow()`，校验内容：

| 校验项 | 说明 |
|--------|------|
| `workflow_data` 存在且为 dict | 结构基本检查 |
| `environment_variables` | 列表 → JSON 转换 |
| `conversation_variables` | 列表 → JSON 转换 |
| `graph` 结构 | 节点/边连接合法性 |
| KNOWLEDGE_RETRIEVAL 节点 | 解密 `dataset_ids`（AES-CBC） |
| `features` 字典 | 可选，格式检查 |
| 节点数据类型 | LLM / TOOL / IF-ELSE 等内部校验 |
| 边连接性 | source/target 合法 |
| 变量引用 | value_selector 指向合法上游 |

**加密处理**: 导出时 dataset_ids 用 AES-CBC 加密（tenant_id 派生 key），导入时解密，解密失败则视为明文 UUID。

## 三、Chat 模式校验链（11 个 Manager）

```python
# ChatAppConfigManager.config_validate()
# 调用顺序:
1.  ModelConfigManager.validate_and_set_defaults(tenant_id, config)
2.  BasicVariablesConfigManager.validate_and_set_defaults(tenant_id, config)
3.  FileUploadConfigManager.validate_and_set_defaults(config)
4.  PromptTemplateConfigManager.validate_and_set_defaults(AppMode.CHAT, config)
5.  DatasetConfigManager.validate_and_set_defaults(tenant_id, AppMode.CHAT, config)
6.  OpeningStatementConfigManager.validate_and_set_defaults(config)
7.  SuggestedQuestionsAfterAnswerConfigManager.validate_and_set_defaults(config)
8.  SpeechToTextConfigManager.validate_and_set_defaults(config)
9.  TextToSpeechConfigManager.validate_and_set_defaults(config)
10. RetrievalResourceConfigManager.validate_and_set_defaults(config)
11. SensitiveWordAvoidanceConfigManager.validate_and_set_defaults(tenant_id, config)
```

### 各 Manager 详解

### 1. ModelConfigManager

| 字段 | 规则 | 错误级别 |
|------|------|---------|
| `model` | 必须存在，必须为 dict | ValueError |
| `model.provider` | 必须存在，必须在可用 provider 列表中 | ValueError |
| `model.name` | 必须存在，必须在该 provider 的 LLM 模型列表中 | ValueError |
| `model.completion_params` | 必须为 dict | ValueError |
| `model.completion_params.stop` | 数组最多 4 项 | ValueError |
| `model.mode` | 自动从模型属性设置，默认 "completion" | 自动修正 |

### 2. BasicVariablesConfigManager

| 字段 | 规则 | 错误级别 |
|------|------|---------|
| `user_input_form` | 默认 `[]`，必须为列表 | ValueError |
| 每项 `.type` | 必须在: text-input, select, paragraph, number, checkbox, external_data_tool | ValueError |
| 每项 `.label` | 必填，字符串 | ValueError |
| 每项 `.variable` | 必填，1-100字符，正则 `^(?!\d)[\u4e00-\u9fa5A-Za-z0-9_\U0001F300-\U0001F64F\U0001F680-\U0001F6FF]+$` | ValueError |
| 每项 `.required` | 可选，默认 false | — |
| select 型 `.options` | 默认 `[]`，`.default` 必须在 options 中 | ValueError |

### 3. FileUploadConfigManager

| 字段 | 规则 |
|------|------|
| `file_upload` | Pydantic `FileUploadConfig` 验证 |
| 默认值 | `image: {enabled: false, number_limits: 1000, detail: "high"}` |

### 4. PromptTemplateConfigManager

| 字段 | 规则 |
|------|------|
| `prompt_type` | 默认 "simple"，可选 "simple" / "advanced" |
| advanced 模式 | 必须有 `chat_prompt_config` 或 `completion_prompt_config` |
| `chat_prompt_config.prompt` | 最多 10 条消息 |
| chat 模型 + completion model | 需要 `user_prefix` / `assistant_prefix` |

### 5. DatasetConfigManager

| 字段 | 规则 |
|------|------|
| `dataset_configs` | 可选 |
| `retrieval_model` | 默认 "single"，可选 "single" / "multiple" |
| dataset ID | 必须为 UUID，必须在 workspace 中存在 |

### 6-11. Features Managers

| Manager | 字段 | 默认值 |
|---------|------|--------|
| OpeningStatement | `opening_statement` (str), `suggested_questions` (list) | `""`, `[]` |
| SuggestedQuestionsAfterAnswer | `enabled` | `false` |
| SpeechToText | `enabled` | `false` |
| TextToSpeech | `enabled`, `voice`, `language` | `false` |
| RetrievalResource | `enabled` | `false` |
| SensitiveWordAvoidance | `enabled`, `type` | `false`；启用时 `type` 必填 |

## 四、Agent 模式校验链（12 个 Manager）

```python
# AgentChatAppConfigManager.config_validate()
1-4.  [与 CHAT 相同: Model, Variables, FileUpload, Prompt]
5.    AgentChatAppConfigManager.validate_agent_mode_and_set_defaults  ← 独有
6.    OpeningStatementConfigManager
7.    SuggestedQuestionsAfterAnswerConfigManager
8.    SpeechToTextConfigManager
9.    TextToSpeechConfigManager
10.   RetrievalResourceConfigManager
11.   DatasetConfigManager   ← 注意：位置在 Features 之后
12.   SensitiveWordAvoidanceConfigManager
```

### Agent 独有校验

| 字段 | 规则 | 错误级别 |
|------|------|---------|
| `agent_mode.enabled` | 布尔值，默认 false | — |
| `agent_mode.strategy` | 枚举: ROUTER / REACT_ROUTER / FUNCTION_CALL | ValueError |
| `agent_mode.tools` | 默认 `[]` | — |
| 新版 tool `.provider_type` | 必填，字符串 | ValueError |
| 新版 tool `.provider_id` | 必填，字符串 | ValueError |
| 新版 tool `.tool_name` | 必填，字符串 | ValueError |
| 新版 tool `.tool_parameters` | 必填，字典 | ValueError |
| 旧版 tool (dataset) | ID 必须为 UUID，数据集必须存在 | ValueError |

## 五、Completion 模式校验链（8 个 Manager）

```python
# CompletionAppConfigManager.config_validate()
1. ModelConfigManager
2. BasicVariablesConfigManager
3. FileUploadConfigManager
4. PromptTemplateConfigManager
5. DatasetConfigManager   ← 特殊：要求 dataset_query_variable
6. TextToSpeechConfigManager
7. MoreLikeThisConfigManager  ← COMPLETION 独有

```

### Completion 独有/差异

| 项目 | 说明 |
|------|------|
| **无** OpeningStatement | 不支持开场白 |
| **无** SuggestedQuestionsAfterAnswer | 不支持回答后推荐问题 |
| **无** SpeechToText | 不支持语音输入 |
| **无** RetrievalResource | 不支持引用来源显示 |
| **有** MoreLikeThis | 独有，`enabled` 默认 false |
| `dataset_query_variable` | 使用知识库时**必填** |

## 七、model_config 完整字段结构

```yaml
model_config:
  model:
    provider: "openai"                    # 必填: provider/name 格式
    name: "gpt-4o"                        # 必填
    mode: "chat"                          # 自动设置
    completion_params:
      temperature: 0.7
      max_tokens: 512
      top_p: 1
      frequency_penalty: 0
      presence_penalty: 0
      stop: []                            # 最多 4 项

  prompt_type: "simple"                   # "simple" | "advanced"
  pre_prompt: "你是一个助手。\n{{query}}"   # simple 模式
  chat_prompt_config: {}                  # advanced 模式
  completion_prompt_config: {}            # advanced 模式

  user_input_form:
    - text-input:                         # | select | paragraph | number | checkbox
        label: "查询"
        variable: "query"                 # 正则: ^(?!\d)[中文A-Za-z0-9_emoji]{1,100}$
        required: true
        default: ""

  dataset_configs:
    retrieval_model: "single"             # "single" | "multiple"
    datasets:
      datasets: []
    metadata_filtering_mode: "disabled"   # "disabled" | "automatic" | "manual"

  dataset_query_variable: ""              # Completion 模式使用知识库时必填

  agent_mode:                             # Agent 模式专属
    enabled: true
    strategy: "function_call"             # ROUTER | REACT_ROUTER | FUNCTION_CALL
    tools:
      - provider_type: "builtin"
        provider_id: "calculator"
        tool_name: "calculate"
        tool_parameters: {}

  # Features（条件可用）
  opening_statement: ""                   # Chat/Agent
  suggested_questions: []                 # Chat/Agent
  suggested_questions_after_answer:
    enabled: false                        # Chat/Agent
  speech_to_text:
    enabled: false                        # Chat/Agent
  text_to_speech:
    enabled: false                        # 全模式
  retriever_resource:
    enabled: false                        # Chat/Agent
  more_like_this:
    enabled: false                        # Completion 独有
  sensitive_word_avoidance:
    enabled: false                        # 全模式
  file_upload:
    image:
      enabled: false
      number_limits: 1000
      detail: "high"
```

## 八、错误类型

所有 Manager 统一抛出 **`ValueError`**，典型消息：

```
"model is required"
"model.provider is required and must be in [...]"
"user_input_form must be a list of objects"
"variable in user_input_form must be a string, and cannot start with a number"
"Dataset ID does not exist, please check your permission."
"Dataset query variable is required when dataset is exist"  (COMPLETION)
"provider_type is required in agent_mode.tools"             (AGENT)
```

部分 Manager 内部使用 Pydantic 验证后转为 ValueError。
